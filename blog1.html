Flex Box In CSS3

Flex box helps to align items and distribute space among container

In flex box items can accomodate(shrink or expand) according to the size of screen

Flex box container: Parent element where we need to apply "display:flex"

Flex items: Child elements inside the flex container

Cross Axis and Main axis

Flex container properties are :

  • Display
  • Flex-direction
  • Flex-wrap
  • Flex-flow
  • Justify-content
  • Align-items
  • Align-content

Display

flex works like block level element it have the same property (Whole-space).

Inline-flex works like Inline-block wraps space according to content.

Flex-direction

Determines the direction of the flex-box

Accepts four different values

Row:Default,item will lay out to left to right.

column:Item will lay out to top to bottom

Row-Reverse:Opposite to "row" direction (right to left)

Column-Reverse:opposite to "column".Item will laid out bottom to top

Flex Wrap

It determines how the flex container will accomodate if there is some extra space in flex container

Three different values

No wrap:Default value,no changes

Wrap:Item will wrapped , flex-items will go to new line if there has no space

Wrap reverse : opposite to wrap felx-items wil in the opposite direction

Flex-flow

Accepts two value at a time

Fisrt: Detremines the flex-direction

Second: determines the flex-wrap

Justify-content property

Property determines how the flex-items will laid ou on the main axis

Helps to make use of extra space left-over in a flex-container

accepts six different values

  • Flex-start
  • Flex-end
  • Center
  • Space-between
  • space-around
  • Space-evenly

Flex-start:Default values items will start from the starting point.

Flex-end:flex-items will start frpm the end point.

Center:flex-items will in the center.

Space-between:The extra space in the flex conatiner will be evenly distributed. the first item will in the container will stick to the starting point of the main axis and last item will attach to the end point.

Space-around:Extra space in the flex container will evenly distributed between flex-items in the "main-axis" suppose there is three boxes so in this case first&last box will take x units space from the wall and it take space x from the center box.

Space-Evenly:In this flex-items will take evenly space from the right and left side both.

Align Items

Similar to justify content,but works on the cross axis and how flex-items align to flex-line

Stretch:Default value flex-items will stretch

Flex-start:Default values items will start from the starting point.

Flex-end:flex-items will start frpm the end point.

Center:flex-items will in the center.

Baseline:According to font-size align to the base line all in the same axis

Align-Content

It works on how the flex-line are align along the cross-axis
